# 📊 Superstore Data Visualization Project
Internship Task 3 – CodeAlpha
👤 Author: Sumitabh Naskar
End-to-end data visualization project on the Superstore dataset using Python, Tableau, and Power BI. Focused on sales trends, profitability analysis, and discount impact, with interactive dashboards and actionable business insights. CodeAlpha Internship Task 3.

📌 Project Overview

This project focuses on exploratory data analysis (EDA) and data visualization using the Superstore dataset. The goal is to uncover key business insights related to sales, profit, discount impact, and customer behavior using multiple tools and technologies.

🎯 Objectives
Analyze sales and profit trends
Identify loss-making products and categories
Understand the impact of discounts on profitability
Build interactive dashboards for business decision-making
🛠️ Tools & Technologies Used
🐍 Python (Pandas, Matplotlib, Seaborn)
📊 Tableau Public
📈 Power BI
📁 Jupyter Notebook

🔍 Key Analysis Performed
📊 1. Sales Analysis
Total sales by category and region
Monthly and yearly sales trends
📉 2. Profit Analysis
Profit distribution across sub-categories
Identification of loss-making products
💸 3. Discount Impact

🔗 4. Correlation Analysis
Sales vs Profit relationship
Discount vs Profit insights
📈 Visualizations Created
🐍 Python (Matplotlib & Seaborn)
Histogram of sales distribution
Scatter plots (Sales vs Profit, Discount vs Profit)
Bar charts (Category-wise sales, Sub-category profit)
Time-series line plots
📊 Tableau Dashboard
KPI Cards (Sales, Profit, Profit Margin)
Monthly Sales Trend
Interactive Filters (Region, Category, Year)
Profit by Sub-category visualization
📈 Power BI Dashboard
Interactive dashboard with slicers
Sales vs Profit scatter analysis
Discount impact visualization
Key Insights panel for storytelling
🔑 Key Insights
📌 Sales show strong growth with seasonal peaks (Q4)
📌 Higher discounts significantly reduce profitability
📌 Sub-categories like Tables and Bookcases contribute to losses
📌 Technology category drives the highest revenue
